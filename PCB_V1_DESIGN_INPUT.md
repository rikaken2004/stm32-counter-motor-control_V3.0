# PCB V1 DESIGN INPUT

> 本文档作为嘉立创 EDA 绘制原理图和 PCB 的设计输入。
> 引脚分配均以当前源代码 (`2-1/user/`, `2-1/hardware/`) 为唯一参考。

---

## 一、PCB V1 定位

| 项目 | 说明 |
|------|------|
| 项目名称 | STM32 智能计数与电机控制系统 —— 模块化载板 V1 |
| 定位 | 模块化载板，不是 STM32 裸芯片板 |
| 核心板 | STM32F103C8T6 现成核心板（Blue Pill 或兼容），通过 2×20 排母插座接入 |
| 电机驱动 | TB6612FNG 现成模块，通过排针/排母插座接入 |
| 其他接口 | OLED、红外传感器、按键、LED、电机、电源、UART 调试全部通过板载接口接入 |
| 目标 | 减少杜邦线，提高接线稳定性和可重复性，为项目演示和简历展示做准备 |

## 四、接口设计表

### 4.1 OLED 接口 (4Pin)

| Pin | 信号 | 连接 | 类型 |
|-----|------|------|------|
| 1 | GND | STM32 GND | 电源 |
| 2 | VCC | 3.3V | 电源 |
| 3 | SCL | PB8 | 信号 |
| 4 | SDA | PB9 | 信号 |

> 注意：不同厂商 OLED 模块 Pin 序可能不同（GND/VCC/SCL/SDA 或 VCC/GND/SCL/SDA），布线前确认实物。

### 4.2 红外传感器接口 (3Pin)

| Pin | 信号 | 连接 | 类型 |
|-----|------|------|------|
| 1 | VCC | 3.3V 或 5V（视模块规格） | 电源 |
| 2 | GND | STM32 GND | 电源 |
| 3 | OUT | PB14 (经 1kΩ 串联电阻) | 信号 |

> 源码使用 `GPIO_IPU`（内部上拉）+ `EXTI_Trigger_Falling`（下降沿触发）。
> 即红外模块输出默认为高电平，检测到物体时拉低。

### 4.3 USART3 调试接口 (4Pin)

| Pin | 信号 | 连接 | 类型 |
|-----|------|------|------|
| 1 | GND | STM32 GND | 电源 |
| 2 | TX | PB10 → USB-TTL RXD | 信号 |
| 3 | RX | PB11 ← USB-TTL TXD（预留） | 信号 |
| 4 | 3.3V | 可选（通常不接，避免供电冲突） | 电源 |

> 接线说明：调试时只接 GND + TX + RX。不要从 USB-TTL 的 3.3V/5V 给 STM32 供电。

### 4.4 TB6612FNG 电机驱动模块接口

> 以下为典型 TB6612FNG 模块引脚。实际布局以购买模块的引脚排列为准。

| 模块引脚 | 连接 | 说明 |
|----------|------|------|
| VM | 电机电源输入端子 (5V~12V) | 电机供电，独立电源 |
| VCC | 3.3V 或 5V | 模块逻辑供电（可取自 STM32 3.3V） |
| GND | STM32 GND + 电机电源 GND | **必须共地** |
| PWMA | PA6 (TIM3_CH1) | 电机 A 通道 PWM 输入 |
| AIN1 | PB12 | 方向控制 1 |
| AIN2 | PB13 | 方向控制 2 |
| STBY | 3.3V 或 VCC | 正常工作需拉高（代码未控制） |

> 源码中 `Motor_SetSpeed()` 的控制逻辑：
> - 正转/停止: PB12=H, PB13=L, PWM 占空比 = Speed%
> - 反转: PB12=L, PB13=H, PWM 占空比 = |Speed|%
>
> 当前代码中 STBY 未被 STM32 控制，需在模块上直接接高电平。

### 4.5 电机输出端子 (2P)

| 端子 | 连接 | 说明 |
|------|------|------|
| MOTOR_A+ | TB6612 AO1 | 直流电机正端 |
| MOTOR_A- | TB6612 AO2 | 直流电机负端 |

### 4.6 电机电源输入端子 (2P)

| 端子 | 连接 | 说明 |
|------|------|------|
| VM | TB6612 VM + 电解电容正极 | 电机电源正极 |
| GND | TB6612 GND + 电解电容负极 + STM32 GND | **共地** |

### 4.7 逻辑电源输入

| 方式 | 说明 |
|------|------|
| STM32 核心板 USB 供电 | 最简单，核心板自带 LDO 输出 3.3V |
| 外部 5V 排针输入 | 接 5V 电源模块，通过核心板 LDO |

> V1 不使用独立 LDO 电路，由核心板或 USB 供电。

### 4.8 按键 (板载)

| 按键 | 连接 | 方式 |
|------|------|------|
| ADD | PB1 → GND | 轻触开关 + 内部上拉 (GPIO_IPU) |
| CONFIRM/RESET | PC13 → GND | 轻触开关 + 内部上拉 (GPIO_IPU) |

> 源码已配置内部上拉，无需外部上拉电阻。但 PCB 上可预留 10kΩ 上拉焊盘。

### 4.9 LED (板载)

| LED | 连接 | 方式 |
|-----|------|------|
| LED9 (计数指示) | PA9 → 限流电阻 → LED → GND | 低电平亮 |
| LED10 (状态指示) | PA10 → 限流电阻 → LED → GND | 低电平亮 |

> 限流电阻建议 220Ω~470Ω（5mm LED，3.3V 供电）。

## 五、电源方案

```
┌─────────────────────────────────────────────────────────┐
│                     电源拓扑 V1                          │
│                                                         │
│  电机电源 (5V~12V)                                       │
│      │                                                  │
│      ├── TB6612 VM (电机驱动供电)                         │
│      ├── 电解电容 100μF~470μF                            │
│      └── 陶瓷电容 0.1μF                                   │
│                                                         │
│  STM32 核心板 USB / 5V                                   │
│      │                                                  │
│      ├── STM32F103 核心板 (板载 LDO → 3.3V)               │
│      └── 3.3V 供给: OLED, 红外传感器, TB6612 VCC          │
│                                                         │
│  ⚠ 两组电源 GND 必须在 PCB 上单点共地                      │
│  ⚠ 不可用 USB-TTL 的 3.3V/5V 给电机供电                   │
│  ⚠ 不可用 ST-LINK 的 3.3V 给电机供电                       │
└─────────────────────────────────────────────────────────┘
```

### 建议

| 项目 | 建议 |
|------|------|
| STM32 供电 | 核心板自带 Micro USB 供电（5V），板载 LDO 输出 3.3V |
| 电机供电 | 独立 5V~6V 电源适配器或电池组 |
| 共地 | 核心板 GND ↔ TB6612 GND ↔ 电机电源 GND ↔ 所有模块 GND |
| 隔离原则 | 电机大电流回路不走 MCU 信号区 |
| 电容位置 | 电解电容靠近 TB6612 VM 引脚；0.1μF 紧贴 VM/GND 引脚 |

## 六、抗干扰设计

> 本章基于已确认的"电机干扰 PB14 红外计数"问题制定。

| # | 措施 | 设计细节 |
|---|------|----------|
| 1 | VM/GND 电解电容 | 100μF~470μF / 16V 或更高，径向引线封装，靠近 TB6612 VM/GND |
| 2 | VM/GND 陶瓷电容 | 0.1μF (104) 0805 或 1206 封装，紧贴 TB6612 VM/GND |
| 3 | 电机两端陶瓷电容 | 0.1μF (104)，直接焊在电机接线端子两端 |
| 4 | PB14 RC 滤波 | 红外 OUT → 1kΩ 串联电阻 → PB14；PB14 端对地 100nF (0805) |
| 5 | PB14 预留上拉 | 10kΩ (0805) 上拉到 3.3V（源码已有内部上拉，外部作为加固） |
| 6 | 红外信号线隔离 | 布线时红外 OUT 走线远离电机 PWM (PA6)、电机方向 (PB12/PB13) |
| 7 | 电机线加粗 | 电机电源输入到 TB6612 VM 的走线，以及 TB6612 到电机端子的走线，宽度 ≥1mm |
| 8 | 电机线位置 | 电机相关走线尽量靠近板边，远离 MCU 区 |
| 9 | GND 铺铜 | 顶层 + 底层大面积 GND 铺铜，减小回路阻抗 |
| 10 | GND 测试点 | 预留 3.3V、5V、VM、GND 测试点，方便示波器查看电源质量 |
| 11 | 信号测试点 | 预留 PB14、PB10 (TX)、PA6 (PWM) 测试点 |

### PB14 RC 滤波电路

```
红外传感器 OUT ──[1kΩ]──┬── PB14 (STM32)
                        │
                      [100nF]
                        │
                       GND
                        │
                      [10kΩ]  (预留上拉，默认不焊)
                        │
                       3.3V
```

> 截止频率 f_c = 1/(2π×1kΩ×100nF) ≈ 1.6kHz，可滤除高频电机噪声。

## 八、PCB 布局建议

### 布局原则

| # | 原则 | 说明 |
|---|------|------|
| 1 | 电源入口靠板边 | 电机电源端子和逻辑电源接口放板边缘，方便接线 |
| 2 | 电机接口靠板边 | 电机输出端子靠板边，电机线不跨越 PCB |
| 3 | TB6612 靠近电机端子 | 缩短大电流路径，VM 和 AO1/AO2 走线尽量短 |
| 4 | STM32 核心板居中 | 作为信号中枢，各模块围绕布局 |
| 5 | OLED 接口放上方或左侧板边 | 方便调整显示角度 |
| 6 | 按键放易操作位置 | 板边区域，手指可按到且不遮挡 OLED |
| 7 | UART 调试口放板边 | 方便插拔 USB-TTL 模块 |
| 8 | 红外接口远离电机区 | 信号走线不平行于电机 PWM/电源走线 |
| 9 | 电机电源走线加粗 | VM、GND、AO1、AO2 走线宽度 ≥1mm |
| 10 | 信号线不走电机电流区 | PB14、PB1、PC13 等信号线不要穿过电机大电流回路 |
| 11 | GND 大面积铺铜 | 顶层 + 底层均铺 GND，过孔阵列缝合 |
| 12 | 电机区与 MCU 区分割 | 铺铜时可在电机区和 MCU 区间用较窄的 GND 桥连接 |

### 建议布局分区

```
┌──────────────────────────────────┐
│  [电机电源]  [电机端子] [TB6612]  │  ← 电机区（右上角）
│                                    │
│           [GND 隔离带]             │
│                                    │
│  [STM32 核心板居中]                │  ← MCU 区（中央）
│  前面: OLED  左侧: 按键+LED       │
│  后面: UART  右侧: 红外           │
│                                    │
│  [电源入口]     [测试点区]         │  ← 左下角
└──────────────────────────────────┘
```
