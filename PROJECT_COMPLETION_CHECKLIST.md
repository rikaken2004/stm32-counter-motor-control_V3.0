## 九、简历项目描述草稿

> 可用于简历「项目经历」栏，建议 4~6 行。

**STM32 智能计数与电机控制系统**

- 基于 STM32F103C8T6 的工业计数控制项目，是 51 双 MCU 方案的 STM32 单芯片工程化升级
- 设计 IDLE / RUNNING / COOLDOWN 三层非阻塞状态机，基于 TIM4 1ms 时基驱动，零 `Delay_ms` 裸机架构
- EXTI 下降沿红外脉冲计数 + 软件消抖，TIM3 PWM 电机调速 (TB6612 驱动)，按键消抖采用 10ms 采样 3 次确认状态机
- USART3 115200bps 事件日志 (启动/状态转换/计数/复位)，用于调试和运行监控
- 定位并整改电机 EMI 干扰红外信号问题：独立供电、RC 滤波、PCB 布局优化
- 使用 Keil MDK + STM32 标准外设库开发，嘉立创 EDA 设计模块化载板 PCB

**英文版 (可选)**：

- STM32F103C8T6-based industrial counting & motor control system; single-MCU upgrade from a dual-8051 design
- Designed a 3-state non-blocking state machine (IDLE→RUNNING→COOLDOWN) driven by TIM4 1ms tick, zero blocking delays
- EXTI-based IR pulse counting with software debounce; TIM3 PWM motor speed control via TB6612 driver
- USART3 115200bps event logging for debugging and runtime monitoring
- Diagnosed and resolved motor EMI interference on the IR sensor signal through isolated power, RC filtering, and PCB layout improvements
- Developed with Keil MDK + STM32 Standard Peripheral Library; modular carrier board PCB designed in EasyEDA
